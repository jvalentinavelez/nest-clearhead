## ClearHead API

1. Clonar proyecto

2. npm install

```bash
$ npm install
```

3. Clonar el archivo `.env.template` y renombrarlo a `.env`
4. Cambiar las variables de entorno
5. Levantar la base de datos postgres

```bash
$ docker-compose up -d
```

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
```
